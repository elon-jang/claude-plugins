Clawdbot 이 맥 미니를 품절시킨 주말, 그리고 카카오가 긴장해야 할 진짜 이유

지난 주말 X 에서 Clawdbot 언급만 해도 좋아요가 1,000개에 육박했어요. ChatGPT 출시 때만큼 반응이 컸습니다.

심지어 Clawdbot 돌리려고 맥미니가 품절되는 상황까지 벌어졌고요. 이미 ChatGPT, Gemini, Claude 가 있는데 왜 이렇게까지 반응이 큰 걸까요?

1️⃣ Clawdbot 이 뭔가요?

Clawdbot 은 개발자 Peter Steinberger 가 PSPDFKit 의 성공적인 Exit(~€ 100M) 이후 다시 복귀해 만든 오픈소스 개인 AI 비서예요. Claude API 를 기반으로 하지만, 핵심은 내 기기에서 로컬로 돌아간다는 거예요. 오래된 노트북, 라즈베리 파이, 게이밍 PC, 맥미니, 저렴한 VPS 어디서든 Node.js 만 있으면 돌릴 수 있어요.

웹 인터페이스가 아니라 WhatsApp, Telegram, Discord, iMessage, Slack 같은 메신저로 대화해요. 마스코트는 우주 랍스터 Clawd 🦞 이고, 슬로건이 "실제로 일을 하는 AI" 예요.

2️⃣ 기존 AI 와 뭐가 다른가요?

기존 AI 는 대화창에서 질문하고 답을 받는 구조였어요. Clawdbot 은 다릅니다.

• 기억이 유지돼요. 이전 대화 맥락을 계속 기억하고 이어가요
• 먼저 말을 걸어요. 아침에 "긴급 이메일 3개, 20분 뒤 회의" 라고 알아서 알려줘요
• 실제로 실행해요. 뉴스레터 구독 취소, 파일 정리, 홈 오토메이션, 웹사이트 수정까지 직접 해요
• 내 기기를 제어해요. 셸 명령어 실행, 웹 브라우징, 캘린더 관리가 다 가능해요

3️⃣ 코드를 뜯어보니 설계 철학이 명확해요

직접 코드를 분석해봤어요. 핵심은 Gateway 라는 중앙 서버가 모든 걸 조율하는 구조예요. 이 Gateway 가 네 가지를 관리해요.

• Sessions: 각 대화의 상태와 맥락을 저장해요. 이게 "기억하는 AI" 의 핵심이에요
• Channels: WhatsApp, Telegram, Slack 같은 메신저 연결을 담당해요
• Agents: Claude 같은 AI 모델이 실제로 사용자 입력을 처리하는 두뇌 역할이에요
• Tools: 브라우저 제어, 파일 관리, 시스템 연동 같은 실행 능력을 제공해요

4️⃣ 메시지가 처리되는 흐름을 보면 왜 강력한지 알 수 있어요

제가 인상 깊었던 건 데이터 흐름이에요. 예를 들어 Telegram 으로 메시지를 보내면 이렇게 처리돼요.

• 채널 플러그인이 메시지를 받아서 표준 포맷으로 변환해요
• Gateway 가 세션을 찾거나 새로 만들어서 맥락을 연결해요
• 에이전트가 대화 히스토리와 시스템 프롬프트를 조합해서 AI 모델에 전달해요
• 모델이 "생각" 하거나 도구를 호출하면, 그 결과가 다시 모델로 피드백돼요
• 최종 응답이 원래 채널로 돌아가고, 연결된 모든 클라이언트에 실시간 동기화돼요

이 구조 덕분에 폰으로 카메라 찍으라고 시키면 에이전트가 실제로 실행할 수 있어요. 맥 앱, iOS 앱, 안드로이드 앱이 전부 "노드" 로 연결되어 있거든요.

5️⃣ 확장성이 진짜 잘 설계되어 있어요

extensions 폴더 구조를 보면 Discord, Matrix, Microsoft Teams 같은 채널이 플러그인처럼 붙어 있어요. 새 메신저 지원하려면 채널 확장만 추가하면 돼요. 도구도 마찬가지고요.

특히 클로드 코드와 마찬가지로 SKILL.md 라는 파일로 커스텀 스킬을 정의할 수 있어요. 프롬프트와 스크립트를 조합해서 나만의 자동화를 만드는 구조예요. 이게 커뮤니티가 빠르게 성장하는 이유 중 하나인 것 같아요.

6️⃣ 이게 왜 ChatGPT, Gemini, Claude 로는 안 됐을까요

단순해요. 비쌌고, 작업 제한이 많았고, 다른 서비스 연결하는 생태계가 부족했어요. Clawdbot 은 로컬에서 돌아가고, 메신저 연동이 기본이고, 셸 명령어까지 실행할 수 있어요. "할 수 있는 일" 의 범위 자체가 달라요.

그리고 결정적으로, 세션 관리 덕분에 맥락이 유지돼요. 기존 AI 는 대화가 끊기면 다시 설명해야 했잖아요. Clawdbot 은 이전 대화를 기억하고 이어가요.

7️⃣ 카카오가 긴장해야 할 지점이 여기 있어요

그 동안 카카오톡 등의 메신저 서비스는 기존의 가입자 네트워크로 강력한 락인 효과를 누려왔습니다. 하지만 AI 로 뭔가를 주문하고 처리하는 데는 사실 내 가족, 내 친구 등 사람이 필요 없어요.

그런데 새로운 AI 비서 메신저가 압도적으로 편하다면? 새로운 메신저의 중심이 생기고, 거기로 사람들이 모일 수 있어요.

다만 Clawdbot 도 비용이 만만치 않다는 것이 문제입니다. 제대로 돌리려면 Claude Opus 급 성능이 필요하거든요.

그런데 DeepSeek V4 가 곧 나올 것 같아요. 최근 논문들 보면 성능도 좋을 거고, 특히 메모리에 특화되어 있어요. 가격까지 저렴하다면? 결말을 예상할 수 없습니다.

8️⃣ 결국 싸움은 "누가 먼저 락인을 만드느냐" 예요

Clawdbot 이 보여준 건 이미 많은 사용자에게 확산된 실제 작업이 가능한 AI 에이전트 인프라에 기반하여 가격 문제가 풀리면 AI 에이전트가 메신저를 대체할 수 있다는 거예요. 개인을 한 명씩 모으고, 그 위에 락인 구조를 쌓으면 게임이 바뀌어요.

메모리를 가진 AI 가 메신저가 되는 시대, 대화의 중심이 사람에서 에이전트로 옮겨가고 있어요.

⚠️ 다만, 시작 전에 꼭 알아두세요

Clawdbot 은 셸 명령어를 실행하고 파일을 직접 다루거든요. 그래서 잘못된 명령이 실행되면 중요한 파일이 삭제되거나 시스템에 문제가 생길 수 있습니다.

그래서 본인의 메인 노트북이나 업무용 기기 대신 별도의 기기를 마련하는 것을 추천드립니다.
