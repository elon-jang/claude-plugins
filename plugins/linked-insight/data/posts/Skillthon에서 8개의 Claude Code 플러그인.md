Skillthon에서 8개의 Claude Code 플러그인이 만들어졌습니다. 1등은 AI 전화로 두쫀쿠 재고를 확인했습니다. 2등은 스킬 사용 통계를 RPG 스타일 캐릭터로 시각화했습니다. 8개의 플러그인을 소개합니다.

1. 두바이쫀득쿠키 리서치 🏆 우승: ElevenLabs AI 전화를 활용해 두바이쫀득쿠키 매장에 직접 전화를 걸어 재고 현황을 자동 확인하는 플러그인. "두쫀쿠 재고 확인해줘" 한 마디로 실행.
https://lnkd.in/gJcw4fMV

2. Skill-Up ⭐ 인기상: 스킬 사용 통계를 RPG 스타일 캐릭터 시트(등급 SSS~F, 레벨, 직업 칭호)로 시각화하고, TDD 기반 압박 테스트로 스킬 품질을 자동 개선하는 플러그인.
https://lnkd.in/geNkGp73

3. 서브웨이 주문 봇: Claude Code가 서브웨이 직원이 되어 팀원들의 샌드위치 주문을 인터랙티브하게 받고, GitHub Issue로 자동 취합해주는 스킬. 3가지 성격 모드(친절/츤데레/프로페셔널)와 6가지 주문 모드(무드픽, 랜덤 가챠, 칼로리 목표 등) 지원.
https://lnkd.in/g_4qzaYZ

4. Task List Manager: Claude Code의 Task를 생성·관리하고 Git으로 추적하여 팀과 공유하는 스킬/훅 모음. Task 이력 조회, 브랜치 간 비교, 변경사항 커밋 등 프로젝트 관리 워크플로우 제공.
https://lnkd.in/gATfC3yQ

5. Visual Workflow Builder: 코드 없이 드래그앤드롭으로 멀티에이전트 워크플로우를 설계하는 노드 기반 비주얼 빌더. Input/Subagent/Skill/MCP/Output 5가지 노드를 연결해 복잡한 AI 파이프라인을 시각적으로 구축·실행.
https://lnkd.in/gFew7i2R

6. Video Editing Skills: FFmpeg 무음 감지와 SRT 자막 분석으로 영상의 불필요한 구간(중복 테이크, 말더듬, 미완성 문장)을 자동 탐지하여 편집 포인트를 제안하는 영상 편집 자동화 플러그인.
https://lnkd.in/g-6-FRVS

7. Auto Check-in: 브라우저 자동화로 다양한 사이트의 출석체크를 AI가 대신 수행하는 플러그인. CSS 셀렉터 대신 자연어 기반으로 요소를 찾아 UI 변경에도 스스로 적응하는 자가 회복(self-healing) 방식.
https://lnkd.in/g24Ufenu

8. Short Video Generator: Story→Image+TTS(병렬)→Review→Video 4단계 파이프라인으로 숏폼 영상을 자동 생성하는 스킬. Tutorial/Review/Story/Listicle/Behind 5가지 유형과 SRT 자막, 인물 일관성 유지 지원.
https://lnkd.in/gQsaPzxw
